# Féria｜出攤筆記 Landing Page

這是 **Féria｜出攤筆記** 的產品發布前預告首頁，使用 Next.js 與 Tailwind CSS 建置，適合部署到 Vercel。

## 品牌定位

Féria｜出攤筆記，是 **手作品牌的市集經營筆記**。目前這不是正式 App，而是 Coming Soon Landing Page，用來在正式版發布前建立期待感、社群導流與內測招募。

## 頁面區塊

- Hero 首屏：產品名稱、核心標語與產品預覽
- 痛點共鳴：出攤品牌常見的紀錄困擾
- 功能預告：市集場次、商品銷售、營收成本、出攤回顧
- 開發進度：產品設計中、內測版本準備中、即將開放試用
- CTA 區塊：追蹤 Instagram / Threads 或等待名單

## 本機開發

```bash
npm install
npm run dev
```

開啟 http://localhost:3000 查看網站。

## 建置

```bash
npm run build
npm run start
```

## 部署到 Vercel

1. 在 Vercel 建立 New Project。
2. 匯入此 GitHub repository。
3. Framework Preset 選擇 Next.js。
4. 按 Deploy 完成部署。
