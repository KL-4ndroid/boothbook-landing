# Féria｜出攤筆記 UI Brand System

## 品牌定位

**Féria｜出攤筆記** 是手作品牌的市集經營筆記。整體視覺應該呈現浪漫、安靜、精緻、留白、有市集感，但仍然是清楚、可信任的工具型 App。

## 核心視覺隱喻

品牌符號來自：

- 市集棚頂
- 筆記紀錄頁面

UI 使用克制的棚頂線條、筆記線條、細線 divider、圓角卡片、霧面紙感背景與柔和陰影。

## 文案規則

- 正式名稱：Féria｜出攤筆記
- 產品定位：手作品牌的市集經營筆記
- 一句話介紹：記錄市集、商品、成本與成果，讓每一次出攤都成為品牌成長的線索。

## 設計判斷

Féria 不是一般記帳 App，而是一個陪手作品牌記下每一次市集成果的經營筆記。
